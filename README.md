# asabbq-legal

Legal pages for the **ASA BBQ** customer mobile app.

Published via GitHub Pages: <https://svlarot.github.io/asabbq-legal/>

The repo is intentionally tiny:

| File | Purpose |
| --- | --- |
| `index.md` | The privacy policy. Served at the root URL above and linked from inside the app. |
| `delete-account/index.md` | Standalone account-deletion instructions. Reachable without signing in, which is what Google Play's data-deletion policy requires. |
| `_config.yml` | Jekyll config (just sets the page title; uses GitHub Pages' default theme). |
| `README.md` | This file. |

To make a change, edit `index.md`, bump the "Last updated" date at the top, and push to `main`. GitHub Pages rebuilds within a minute or two.
